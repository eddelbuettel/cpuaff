cpuaff

Multi-platform C++11 interface to CPU affinity management

Download this project as a .zip file Download this project as a tar.gz file

Description

cpuaff is a C++11 library that abstracts CPU affinity settings for multiple platforms. It aims to be easy to use yet powerful.

Releases

To get a stable release, go to the downloads page.

Details

cpuaff builds up a four part view of every processor on a system. CPUs are identified by their socket, core, and processing_unit. Additionally each cpu on the system has a notion of its NUMA node. This allows a developer to select the CPUs at runtime that achieve the performance goals of his/her application.

cpuaff aims to eventually be completely header-only for easy integration into a system. For the time being the Linux implementation is header-only. Other platforms are implemented using hwloc as an interface to the underlying hardware.

Rationale

Managing the CPU affinity of threads within a process is a powerful tool, but operating system provided interfaces are often difficult to use and hard to understand. To this end, I have created cpuaff which will give an abstract view of the CPUs on a given machine and an easy to use interface to get/set them within a process.
